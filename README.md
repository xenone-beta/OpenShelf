# OpenShelf

OpenShelf — это лёгкий фронтенд-прототип платформы для совместной разработки в стиле GitHub.
Проект реализован как статическое SPA-приложение на чистых `HTML + CSS + JavaScript` без backend-зависимостей.

## Возможности

- Тёмная тема интерфейса.
- Навигация по вкладкам:
  - Overview
  - Repositories
  - Issues
  - Pull Requests
  - Profile
- Поиск по репозиториям.
- Фильтр репозиториев по языку.
- Модальные формы для создания:
  - Issue
  - Pull Request
- Empty-state экраны без фейковых пользователей и искусственной активности.

## Структура проекта

- `index.html` — основная разметка SPA и модальные окна.
- `styles.css` — стили интерфейса и адаптивность.
- `app.js` — клиентская логика (рендер, табы, фильтр, формы).

## Быстрый старт

### 1) Клонирование

```bash
git clone <repo-url>
cd OpenShelf
```

### 2) Запуск локального сервера

```bash
python3 -m http.server 4173
```

### 3) Открытие в браузере

Откройте:

```text
http://localhost:4173
```

## Как использовать

1. Переключайтесь между разделами через верхнее меню.
2. Используйте строку поиска и фильтр языка в секции Repositories.
3. Создавайте Issue и Pull Request через кнопки в левом блоке Quick actions.

## Ограничения текущей версии

- Нет backend/API.
- Нет авторизации и реальных пользователей.
- Данные не сохраняются между перезагрузками страницы.

## Дальнейшее развитие

- Интеграция с backend (REST/GraphQL).
- Подключение базы данных и постоянного хранилища.
- Ролевая модель и система авторизации.
- Реальные уведомления и история активности.
